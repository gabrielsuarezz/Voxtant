# Claude Code Project Preferences

## Git Commit Rules
- **NO** Co-Authored-By line in commits
- **NO** "Generated with Claude Code" attribution
- User should be the only contributor listed on GitHub

## Development Style
- **NO** emojis in code, commits, or any development work unless specifically requested
- Keep all development professional and emoji-free by default

## Additional Notes
- These preferences apply to all work in this project
- Only deviate from these rules when explicitly instructed
